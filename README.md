# Пример использования swarm.js 

Для запуска:

    // Установка browserify
    npm install -g browserify
    // Установка зависимостей
    npm install
    // Сборка бандла для браузера
    npm run rebild
    // Запуск сервера
    npm run start
    // Откройте index.html в браузере
    open index.html
